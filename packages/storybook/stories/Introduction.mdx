# Void UI

A component library built on Lit 3 web components with a strict theme-separation architecture.

## Packages

- `@voidable/theme` — CSS custom properties and design tokens
- `@voidable/ui` — Lit web components (Light DOM, zero visual CSS)

## Theme Separation

Components in `@voidable/ui` ship with **zero visual CSS**. All styling flows through `--vdb-*` CSS custom properties defined by `@voidable/theme`.

Without a theme loaded, components are completely unstyled.
